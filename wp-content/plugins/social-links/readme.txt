== Description ==



Social Links is a sidebar widget that displays icon links to your profile pages on other social networking sites.



**Supported Networks:**

* Blogs
* BlogMarks

* Delicious

* Picasa Web Albums
* Plurk
* Pownce
* Sixent
* Stumble Upon
* Technorati
* Twitter
* YouTube
* Xing



== Installation ==



1. Download the installation zip file and unzip leaving the directory structure in tact.
1. Upload the newly unzipped 'social-links' folder to the `/wp-content/plugins/` folder
1. Activate the Social Links plugin from the 'Plugins' menu.

1. Add the widget to your sidebar from the 'Widgets' design page.

1. Add your social links from the 'Social Links' page under the 'Management' page.
