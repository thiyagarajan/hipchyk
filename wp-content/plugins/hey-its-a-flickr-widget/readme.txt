Easily display recent images from your Flickr photostream with this lightweight, fast plugin that you can control with a sidebar widget. No API keys!


== Description ==
Hey It's a Flickr Widget is, well, a Flickr widget. Simply install it like any other plugin and then use the widget it creates however you'd like. You have total control over the width of the generated images, what account is being accessed, and how many images you'd like to display.
No API keys needed!




== Installation ==
1. Upload `hey-its-flickr.php` to `wp-content/plugins`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to `Appearance > Widgets` and set up the widget wherever you have an active sidebar or widget area!

All versions 2.8 and up have been tested and are supported.

No.
