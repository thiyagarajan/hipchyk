== Installation ==

* Unzip the comments-likes plugin in the plugins folder in wp-content directory i.e `wp-content/plugins`.
* Go to your wordpress site admin and plugins section and activate the comments-likes plugin

== Other Notes ==

* To customize the like button, make changes to the comments-likes.css file in the plugin folders
